# b站更换gif头像教程

链接失效请 h477959413@163.com 进行维护

主要流程：
找到喜欢的gif->裁剪为180×180尺寸，压缩为1MB以内大小->转换为webp格式->
PC端浏览器安装油猴脚本->安装上传gif脚本->上传刷新即可
